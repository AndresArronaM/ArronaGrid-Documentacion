ArronaGrid

V0.1.3

Libreria sass para la maquetacion de sitios web

Descargar GitHub

¿Que es?

ArronaGrid es una libreria sass(con version css) para la maquetacion de sitios web

Solo Maquetacion

No añade colores,tipografias, ni ningun otro estilo visual

Lo Ultimo del estandar

ArronaGrid utiliza lo ultimo del estandar css

Instalacion

Desde GitHub

Descarge la version estable o clone el proyecto desde github

                            
                                    git clone https://github.com/AndresArronaM/ArronaGrid.git

Version sass

importe el archivo ArronaGrid.scss a su proyecto

                            
                                    @import "arronagrid/ArronaGrid.css";

Version CSS

Coloque el linken sus archivos html

                            
                                    <link rel="stylesheet" href="arronagrid/css/ArronaGrid.css">

Desde NPM

Ejecute este comando desde una terminal

                            
                                    npm install --save @andresarronamontoya/arronagrid

Version sass

importe el archivo ArronaGrid.scss a su proyecto

                            
                                    @import "node_modules/@andresarronamontoya/arronagrid/ArronaGrid.scss";

Version CSS

Coloque el linken sus archivos html

                            
                                    <link rel="stylesheet" href="node_modules/@andresarronamontoya/arronagrid/css/ArronaGrid.css">

Configuracion

Gulp

Configuracion de Gulp

Crear el archivo gulpfile.js

E importar el proyecto

                            const gulp = requiere('gulp');
                       const sass = requiere('gulp-sass');
                       const arronagrid = "node_modules/@andresarronamontoya/arronagrid";

                       gulp.task('styles', ()=>{
                            gulp.src('./Style.scss')
                            .pipe(sass({includePaths: [arronagrid]}))
                            .pipe(gulp.dest('./css'))
                       });

Puede complementar esta configuracion de gulp con otros paquetes de NPM

Recomendaciones

autoprefixer Para colocar los prefijos a las propiedades no soportadas por algunos navegadores

cssnano Para minificar los estilos

Nota 1:autoprefixer necesita otro paquete de NPM para funcionar y es Autoprefixer-Core Nota 2:estos paquetes son plugins de PostCss para poder usarse es nesesario instalar el paquete gulp-postcss

browser-sync Genera un servidor local de archivos desde NodeJs

gulp-plumber Para cachear errores y que no se interrumpa las tareas de gulp

gulp-sourcemaps

gulp-watch Para observar nuevos archivos

holderjs Para generar imagenes de maquetacion

                            t:  768px,

Layout

Descripcion

Mixin para determinar el modelo de layout a usar

Parametros

Variable Descripcion Valores
$Display Determina el modelo de layout a usar

flex

grid

sass

                            
                                .content{
                                    @mixin Layout($Display);
                                }

Compila a

                             .content{
                                display:-ms-flexbox;
                                display:flex;
                                -ms-flex-wrap:wrap;
                                flex-wrap:wrap;
                        }

                        //O tambien compila a

                        .content{
                                display:-ms-grid;
                                display:grid
                        }

css

                            <div class="ag-layout-flex"> <div>
                       <div class="ag-layout-grid"> <div>

MaxWidth

Variable Descripcion Valor
$Max-Width Determina tamaño maximo de un item o content 1200px
$FullLayout Determina tamaño maximo de un item o content en base al viewport 100vh
$Full Determina tamaño maximo de un content 100%

                        .content{
                            max-width:$Max-Width;
                    }

Buttons

Buttons

Descripcion

Mixin que le dan los estilos basico para un boton
Variable Descripcion Valor
$Gap Variable que determina el espacio entre los elementos 1rem

sass

                            .boton{
                                @include button($Gap)
                        }

Compila a

                            .boton{
                                display:inline-block;
                                padding:.8em 1.4em;
                                cursor:pointer;
                                border:none;
                                text-decoration:none;
                                margin-right:1rem;
                                margin-left:1rem;
                        }

css

                        
                            <div class="ag-button"> </div>

css

Grid

Descrippcion

Mixin para la creacion de un sistema de rejillas

variable Descripcion valor
Column Varible para determinar el numero de columnas 12

sass

                        .container{
                            @include gridStrict(12);
                    }

compila a

                            -ms-grid-columns: (1fr)[12];
                            grid-template-columns: repeat(12, 1fr);
                    }

css

                        <div class="ag-container-grid ag-Grid-xs">
                            <div class="item"></div>
                    </div>
